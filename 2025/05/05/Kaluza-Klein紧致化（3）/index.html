Kaluza-Klein紧致化（3）

发表于 2025-05-05
该系列的此部分翻译自一篇不知名的讲义。这篇讲义是我见过的关于KK紧化这一主题最亲民的入门材料。

\[ \newcommand{\dd}{\mathrm d} \]

44.3 规范不变性的起源

至少在物理中，奇迹需要一个合理的解释。那么让我们尝试基于先验的理由去理解为什么会出现 Kaluza-Klein 奇迹。对此，让我们回想 Kaluza 对于线元的拟设\((44.4)\)： \[ \begin{aligned} \dd{\hat s}^2_{KK}=g_{\mu\nu}(x^\lambda)\dd x^\mu\dd x^\nu+(\dd x^5+A_\mu(x^\lambda)\dd x^\mu)^2 \end{aligned} \tag{44.17} \] 并将其与五维线元的最一般形式对比，即 \[ \begin{aligned} \dd{\hat s}^2 &\ \ =\ \ \hat g_{MN}(x^L)\dd x^M\dd x^N\\ &\ \ =\ \ \hat g_{\mu\nu}(x^\lambda,x^5)\dd x^\mu\dd x^\nu+2\hat g_{\mu5}(x^\lambda,x^5)\dd x^\mu\dd x^5+\hat g_{55}(x^\mu,x^5)(\dd x^5)^2 \end{aligned} \tag{44.18} \] 很明显，一般的五维线元\((44.18)\)在任意五维坐标变换\(x^M\to\xi^{M'}(x^N)\)下是不变的。然而，这对于 Kaluza-Klein 的拟设\((44.17)\)​不成立。

然而，Kaluza-Klein线元的形式在以下两类坐标变换下是不变的：

  1. 一种是四维的坐标变换 \[ \begin{aligned} x^5&\ \ \to\ \ x^5\\ x^\mu&\ \ \to\ \ \xi^{\nu'}(x^\mu) \end{aligned} \tag{44.19} \] 正如我们所知道的，在这种变换下，\(g_{\mu\nu}\)以保持\(g_{\mu\nu}\mathrm dx^\mu\mathrm dx^\nu\)不变的方式变换，\(A_\mu=\hat g_{\mu5}\)像一个四维对偶矢量一样变换，因此\(A_\mu\mathrm dx^\mu\)​是不变的，并且因此整个度规是不变的。

  2. 还剩一种是五维的坐标变换 \[ \begin{aligned} x^5 & \ \ \to\ \ \xi^5(x^\mu,x^5)=x^5+f(x^\mu)\\ x^\mu & \ \ \to\ \ \xi^\mu(x^\nu)=x^\mu \end{aligned} \] 在这个变换下，\(g_{\mu\nu}\)和\(g_{55}\)都是不变的，但\(A_\mu=g_{\mu5}\)变成： \[ \begin{aligned} A'_\mu &=\hat g'_{\mu 5}=\dfrac{\partial x^M}{\partial \xi^\mu}\dfrac{\partial x^N}{\partial \xi^5}\hat g_{MN} \\ &=\dfrac{\partial x^M}{\partial x^\mu}g_{\mu5}\\ &=g_{\mu 5}-\dfrac{\partial f}{\partial x^\mu}g_{55}\\ &=A_\mu-\partial_\mu f \end{aligned} \tag{44.21} \] 换句话说，Kaluza-Klein线元在平移\(x^5\to x^5+f(x^\mu)\)配上\(A_\mu\to A_\mu-\partial_\mu f\)的变换下是不变的（并且这当然也能直接从度规中读出）。

这恰好就是矢势\(A_\mu\)​的规范变换，并且在当前的语境下，我们发现规范变换作为五维广义协变的产物而出现。

现在已经很清楚我们一定会得到四维的爱因斯坦-麦克斯韦作用量了：首先，由于对\(x^5\)的积分，\(x^5\)的平移是无关紧要的。并且从五维爱因斯坦-希尔伯特作用量出发，我们最终必定能得到一个依赖于\(g_{\mu\nu}\)和\(A_\mu\)的四维作用量，其：

  1. 广义协变（在四维意义下），
  2. 包含到二阶导数，
  3. 并且在\(A_\mu\)的规范变换下不变。

那么唯一的可能就是爱因斯坦-麦克斯韦作用量。

一个考察规范对称性的起源的富有成效的方法，是将其作为“在\(x^5\)上移动一个常数（我想这里的constant作者是指不依赖于\(x^5\)？）是等度规”的一个结果，即\(\partial/\partial x^5\)是度规\((44.17)\)的一个Killing矢量场。\(x^5\)方向上的“内部的”圆周的等度规群即\(SO(2)\)变成了四维理论的规范群\(U(1)=SO(2)\)。

从这个视角来看，矢势的规范变换起源于\(\hat g_{\mu5}\)沿矢量场\(f(x^\mu)\partial_5\)的李导数： \[ \begin{aligned} Y=f(x^\mu)\partial_5\ &\implies\ Y^\mu=0\\ &\ \ \ \ \ \ \ \ \ \ \ Y^5=f\\ &\implies\ Y_\mu=A_\mu f\\ &\ \ \ \ \ \ \ \ \ \ \ Y_5=f\\ (L_Y\hat g)_{\mu5}\ &= \ \hat\nabla_\mu Y_5+\hat\nabla_5Y_\mu\\ &= \ \partial_\mu Y_5-2\hat\Gamma^\mu_{5M}Y^M\\ &= \ \partial_\mu f+{F^\nu}_\mu Y_\nu+F_{\mu\nu}A^\nu Y_5\\ &= \ \partial_\mu f\\ &\iff \delta A_\mu=-\partial_\mu f \end{aligned} \] 这个观点在人们想要通过这个方式得到非阿贝尔规范对称性的时候变得尤为重要：人们从带有等度规群\(G\)的更高维内部空间出发，并为度规做类似的拟设。那么在更高维的一般的坐标变换中，仍有依赖于\(x^\mu\)的内部空间的等度规。这些作用类似于度规的非分块对角部分的非阿贝尔规范变换，并且在内部空间积分时，可以保证得到（也许还有其他东西）四维爱因斯坦-希尔伯特以及杨-米尔斯作用量。
