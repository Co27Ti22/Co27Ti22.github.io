数学物理方法笔记

发表于 2021-05-03 更新于 2021-06-28
没什么好写的，不过是学习笔记而已。不过话说回来，将数学应用于物理，这可是我学物理的初心啊。。

数学物理方法的课程包括两部分：

第一部分为复变函数，第二部分为数理方程。

复变函数论

复数的基本理论

复数的表示法：

  • 通常 \[ z = x + yi \]

  • 指数表示 \[ \begin{aligned} z &= (\rho, \varphi)\\ &=\rho\cos\varphi+i\rho\sin\varphi \quad由欧拉定理，e^{ix} = cosx + isinx\\ &=\rho e^{i\varphi} \end{aligned}复数的几何意义： \]

复数的共轭：

\[ z = x + yi的共轭为\bar{z} = x - yi \]

通过联立\(z\)与\(\bar{z}\)，可以把\(x,y\)用\(z\)与\(\bar{z}\)表示出来。 \[ \begin{cases} x = \dfrac{z + \bar{z}}{2}\\ y = \dfrac{z - \bar{z}}{2i} \end{cases} \]

复数的几何意义：

  • 复向量 \[ z = (x, y) \]

  • 极坐标 \[ z = (\rho, \varphi) \]

  • 球极投影

模长，辐角

\[ \begin{aligned} &|z| = \rho = \sqrt{x^2+y^2}\\ &\varphi = Argz \in(-\infty, +\infty)\\ &argz\in(-\pi, +\pi]\\ &\varphi = argz + 2k\pi \end{aligned} \]

其中，\(\varphi\)为复数z的辐角。由于z的辐角存在多解性，因此定义\(argz\)为辐角的主值。

对于给定的复数，其辐角主值（取值为\((-\pi, +\pi]\)）和\(arctan\frac{y}{x}\)（取值为\([-\frac{\pi}{2}, +\frac{\pi}{2}]\)）有关系： \[ argz = \begin{cases} arctan\frac{y}{x} &(x, y)\in 第一象限\\ arctan\frac{y}{x} + \pi &(x, y)\in 第二象限\\ arctan\frac{y}{x} - \pi &(x, y)\in 第三象限\\ arctan\frac{y}{x} &(x, y)\in 第四象限\\ \end{cases} \]

由此，可以把曲线\(f(x, y) = 0\)用复数\(z, \bar{z}\)来表示。

复变函数

复变函数，与实变函数相对应，即复变量函数，自变量和因变量都为复数的函数。

\(w=f(z)\)

复变函数与实变函数一样，都要求有单值性（一一对应）。每个复变函数由两个二元实函数唯一确定。 \[ z = x + iy\\ f(z)=u(x,y)+iv(x,y) \]

复变函数单值性 黎曼曲面

考察这样的一个复变函数： \[ w=\sqrt z \] 将自变量\(z\)写成指数表示： \[ w=\sqrt \rho e^{i(\frac{\theta}{2}+n\pi)} =\begin{cases} \sqrt \rho e^{i\frac{\theta}{2}} &n=0,2,4,6...\\ -\sqrt \rho e^{i\frac{\theta}{2}} &n=1,3,5,7... \end{cases} \] 当\(n\)分别取奇/偶的时候，我们发现，由于辐角的多值性，函数对应了不同的值。

为了保证复变函数的单值性要求，我们必须对\(n\)为奇偶的情况分别考虑。

支点

当自变量z围绕某一点z_0转一圈回到原点处函数值改变了，z_0即为函数的支点。

\(w=\sqrt z\)支点为：\(z_1=0,z_2=\infty\)

\(w=\sqrt {z-a}\)支点为：\(z_1=a,z_2=\infty\)

在两个支点间连接一条割线，这条割线表示从某处开始，绕支点旋转一周的过程中，不是回到原处，而是在割线处跳到了另一个黎曼面。

所以，对于一个复变函数的定义域，更准确的说法是定义在函数对应的黎曼面上而不是复平面上。

通过构造黎曼面来区分开不同辐角处的同一复数，我们使复变函数也满足了单值性的要求。

解析函数

解析函数是一类特殊的复变函数。有许多优秀的性质。是复变函数论中主要研究的函数。

  定义（复变函数连续性）：\(f(z)\)在\(z_0\)点及其邻域上有定义，且当沿任何路径\(z\rightarrow z_0\)时都有\(\lim_{z\rightarrow z_0} f(z)=f(z_0)\)，则\(f(z)\)在\(z_0\)处连续。

  定义（复变函数可导性）：\(f(z)\)在\(z_0\)处连续，且当以任何方式\(\Delta z\rightarrow 0\)时都有\(f'(z_0)\equiv\lim_{\Delta z\rightarrow 0} \dfrac{f(z_0+\Delta z)-f(z_0)}{\Delta z}\)存在且唯一，则\(f'(z_0)\)即为函数在\(z_0\)处的导数。

  定义（复变函数解析性）：\(f(z)\)在\(z_0\)点及其邻域上各点均可导，则\(f(z)\)在\(z_0\)点解析。

  定义（解析函数）：\(f(z)\)在区域\(D\)上处处解析，则\(f(z)\)为区域\(D\)上的解析函数。

解析函数的判定

既然解析函数有良好的性质，那么我们自然希望有方便的方法来判定一个函数是否是一个解析函数。

  定理（柯西-黎曼条件）：若函数\(f(z)=u(x,y)+iv(x,y)\)在区域\(D\)上可导，则在区域\(D\)上必有下述等式成立： \[ \begin{cases} \dfrac{\part u(x,y)}{\part x} = \dfrac{\part v(x, y)}{\part y}\\ \dfrac{\part u(x,y)}{\part y} = -\dfrac{\part v(x, y)}{\part x}\\ \end{cases} \]

上述定理给出了函数可导（解析）的一个必要条件。

柯西-黎曼条件的等式本质上就是沿着平行于x轴方向和平行于y轴方向通过定义求函数导数并相等推出的

  定理（可导的充要条件）：若函数\(f(z)=u(x,y)+iv(x,y)\)的实部和虚部均为二元可微实函数，并且在区域\(D\)上满足柯西-黎曼条件，则函数\(f(z)\)在区域\(D\)上可导。

  证明：

上述定理给出的是可导/解析的充要条件

解析函数的性质

上面我们一直在说解析函数有许多好的性质，但是到目前为止我们一个都没提，一下将介绍一些简单的解析函数特性：

性质一：解析函数的实部与虚部之间有联系

由解析函数的柯西-黎曼条件即可看出。事实上，由柯西-黎曼条件，只要知道解析函数的实部或虚部以及某一点的值，便可以确定整个解析函数。

性质二：解析函数的实部和虚部为满足拉普拉斯方程的调和函数。

  证明：从柯西-黎曼条件出发 \[ \begin{cases} \dfrac{\part u(x,y)}{\part x} = \dfrac{\part v(x, y)}{\part y}\\ \dfrac{\part u(x,y)}{\part y} = -\dfrac{\part v(x, y)}{\part x}\\ \end{cases} \] 两个方程同时对x求二阶偏导，并作和。在同时对y求二阶偏导，作差即得。

上述性质也可以用于解析函数的判定。

性质二：解析函数与自变量的复共轭无关。

函数\(f(z)=u(x,y)+iv(x,y)\)可以看成是关于\(x,y\)的二元函数。

我们之前了解过\(\begin{cases} x = \dfrac{z + \bar{z}}{2}\\ y = \dfrac{z - \bar{z}}{2i} \end{cases}\)。因此，一般的复变函数函数\(f(z)\)可以通过代换，变成用\(z\)和\(\bar z\)来表示。

而解析函数的一个特点就是与自变量的复共轭无关。

  证明：将\(f(z)\)看成是关于\(x,y\)的二元函数，关于\(z,\bar z\)的复合函数。对\(\bar z\)求偏导，发现为0

解析函数的确定

之前已经提到过，解析函数的实部与虚部之间存在联系。因此，在已知解析函数实部或虚部以及一点的取值的情况下，就可以确定整个解析函数。

以下以已知实部为例：

法一：由柯西-黎曼条件： \[ \begin{cases} \dfrac{\part u(x,y)}{\part x} = \dfrac{\part v(x, y)}{\part y}\\ \dfrac{\part u(x,y)}{\part y} = -\dfrac{\part v(x, y)}{\part x}\\ \end{cases} \]

最大模定理

  定理：设复变函数\(f(z)\)在闭区域上解析，则该复变函数的模\(\mid f(z)\mid\)的最大值只能出现在区域边界上，除非\(f(z)\)是常数。

复变函数的幂级数展开

  定理：设复变函数\(f(z)\)在以为圆心的的圆域内解析，则对于该圆域内任一点，有 \[ \begin{aligned} &f(z)=\sum_{n=0}^{\infty}a_n(z-z_0)^n\\ &a_n=\dfrac{1}{2\pi i}\oint_C \dfrac{f(\xi)}{(\xi-z_0)^{n+1}}d\xi = \dfrac{f^{(n)}(z_0)}{n!} \end{aligned} \]
